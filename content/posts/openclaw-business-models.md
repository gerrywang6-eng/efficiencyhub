# skills/ecommerce-scraper/main.py

class EcommerceScraper:
    def __init__(self):
        self.platforms = ['amazon', 'shopify', 'woocommerce']
    
    def scrape_product_data(self, url):
        # 使用OpenClaw控制浏览器
        openclaw.browser.goto(url)
        data = openclaw.browser.extract({
            'title': 'h1',
            'price': '.price',
            'rating': '.rating'
        })
        return data

### 常见陷阱

**陷阱1：追求完美**
- 问题：花3个月做产品，没验证需求
- 解决：2周出MVP，快速验证

**陷阱2：定价过低**
- 问题：$50/月，无法盈利
- 解决：至少$200/月，筛选优质客户

**陷阱3：服务太广**
- 问题：什么都能做，什么都不精
- 解决：聚焦一个垂直领域

**陷阱4：忽视留存**
- 问题：只关注获客，客户流失率高
- 解决：客户成功比销售更重要

**陷阱5：单打独斗**
- 问题：一个人做所有事
- 解决：尽早建立团队或合作

### 成功要素

**必备素质**：
1. **执行力**：想到就做，快速迭代
2. **学习能力**：技术更新快，持续学习
3. **客户导向**：解决真实问题
4. **商业思维**：不只关注技术
5. **长期主义**：不追求短期利益
